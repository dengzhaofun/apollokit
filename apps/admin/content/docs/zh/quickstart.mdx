本文档会带你在 5 分钟内走完「注册 → 建组织 → 签发客户端凭证 → 调通第一个端点」的完整路径,随后再跳去各模块详细文档。

占位:详细内容将在下一次提交里补上。当前先介绍目录骨架。

## 流程

1. 注册管理端账号并登录。
2. 在控制台创建一个 organization。
3. 进入「API Keys」为该 org 签发一把管理端 API key。
4. 进入「Client Credentials」签发一对 clientId + secret 给游戏客户端。
5. 用 cURL 调一次 `/api/check-in` 验证链路。
