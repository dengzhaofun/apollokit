本页从**运营视角**梳理 ApolloKit 所有模块能力。每项后面标注：

- ✅ **管理台自助** — 无需开发，管理台直接配置
- ⚡ **AI 一句话配置** — 支持 AI 代理自动填表，自然语言描述即可完成

---

## 经济体系

游戏内的金钱与资产流通，覆盖从发放到消耗的全链路。

| 功能 | 描述 | 自助程度 |
|------|------|---------|
| **货币系统** | 定义多种虚拟货币（金币、钻石、积分……），管理发放、扣减、余额和流水审计 | ✅ ⚡ |
| **物品与背包** | 物品定义、分类、背包管理、道具发放与审计 | ✅ ⚡ |
| **商店** | 商品分类、标签、成长包阶段、限购规则、购买链路 | ✅ ⚡ |
| **兑换码** | 批次生成、单码/单人/全局限制、KOL 投放 | ✅ ⚡ |
| **积分兑换** | 积分或物品换奖励，支持单人与全局限额，幂等执行 | ✅ |
| **存储箱** | 储蓄 + 利息玩法，支持定期储蓄 / 风险投资等机制 | ✅ ⚡ |

## 运营活动

玩家留存与促活的核心工具，全部支持管理台独立配置与上线。

| 功能 | 描述 | 自助程度 |
|------|------|---------|
| **签到** | 每日签到、连续签到追踪、周期自动重置、多级奖励配置 | ✅ ⚡ |
| **任务系统** | 任务分类与定义、事件驱动进度更新、多级阶梯奖励 | ✅ |
| **抽奖** | 奖池定义、档位分层、奖品配置、保底机制、单抽与多抽 | ✅ ⚡ |
| **活动** | 生命周期管理、节点与时间调度、积分 + 里程碑奖励 | ✅ |
| **系统邮件** | 系统邮件发送、附件奖励（货币/物品）、玩家端领取 | ✅ ⚡ |

## 公告与触达

精准触达玩家，提升活动曝光与转化。

| 功能 | 描述 | 自助程度 |
|------|------|---------|
| **弹窗公告** | 弹窗 / 列表两种形式，支持定向投放，追踪曝光与点击 | ✅ ⚡ |
| **轮播 Banner** | 分组 + 轮播图 + CTA 跳转 + 时间窗控制 | ✅ ⚡ |

## 社交玩法

提升玩家粘性与社群归属感的社交模块。

| 功能 | 描述 | 自助程度 |
|------|------|---------|
| **好友系统** | 好友关系、申请、拒绝、好友礼物发放 | ✅ |

## 内容管理

游戏叙事与关卡内容的结构化管理，配合运营活动灵活下发。

| 功能 | 描述 | 自助程度 |
|------|------|---------|
| **对话脚本** | 对话树定义，支持分支与条件 | ✅ |
| **关卡配置** | 关卡参数管理，服务端配置化 | ✅ |
| **实体** | 通用游戏实体定义（NPC、地图点位等） | ✅ |
| **媒体库** | 图片、视频、资源 URL 的集中管理 | ✅ |
| **合集** | 内容分组与合集编排 | ✅ |
| **链接** | 跳转链接与 CTA 目标管理 | ✅ |
