## 解决什么问题

每款游戏都要一套「金币 / 钻石 / 体力 / 能量」式的内部代币账本。货币模块提供:

- 多种货币类型定义(允许同一组织配置多种代币)
- 发放与扣减的原子操作(并发安全)
- 变更流水与余额查询
- 与 [shop](./shop)、[exchange](./exchange)、[mail](./mail) 等模块配合形成完整经济闭环

更详细的字段、API 形态请参考 `/openapi.json` 中 `currency` 相关端点,或等待本页补齐。
