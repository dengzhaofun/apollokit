## Two user identities

- **Admin user** — authenticated via Better Auth, owns an Organization context.
- **End user** — the integrator's own business user, identified by an opaque `endUserId`. ApolloKit does not manage their identity system.

See [organizations](./organizations) and [authentication](./authentication).

## Error envelope

All failures return:

```json
{
  "error": "human readable",
  "code": "module.reason",
  "requestId": "req_01hx..."
}
```

Branch on `code`; look up logs by `requestId`. Full list at [errors](./errors).
