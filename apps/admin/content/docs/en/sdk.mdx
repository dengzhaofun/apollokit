`@apollokit/admin` is a TypeScript SDK auto-generated from the OpenAPI spec, offering full type safety for admin API calls.

## Install

```bash
npm install @apollokit/admin
# or
pnpm add @apollokit/admin
```

> Not yet published to npm. Consume via workspace reference or build from source.

## Initialize

```typescript
import { createAdminClient } from '@apollokit/admin';

const client = createAdminClient({
  baseUrl: 'https://api.your-domain.com',
  apiKey: 'ak_your_admin_api_key',
});
```

`createAdminClient` automatically injects `x-api-key` on every request.

## Usage examples

### Check-in module

```typescript
// Create a check-in config
const config = await postApiCheckInConfigs({
  body: {
    name: 'Daily Check-in',
    alias: 'daily',
    resetMode: 'week',
    weekStartsOn: 1,
    target: 7,
    timezone: 'Asia/Shanghai',
    isActive: true,
  },
});

// List all configs
const configs = await getApiCheckInConfigs();

// Fetch a config by UUID or alias
const daily = await getApiCheckInConfigsByKey({
  path: { key: 'daily' },
});

// Configure day-3 rewards
await postApiCheckInConfigsByKeyRewards({
  path: { key: 'daily' },
  body: {
    dayNumber: 3,
    rewardItems: [
      { definitionId: 'gold-uuid', quantity: 100 },
    ],
  },
});

// Execute check-in
const result = await postApiCheckInConfigsByKeyCheckIns({
  path: { key: 'daily' },
  body: { endUserId: 'user-42' },
});
```

### Item module

```typescript
// Create an item definition
await postApiItemDefinitions({
  body: {
    name: 'Gold',
    alias: 'gold',
    stackable: true,
    isActive: true,
  },
});

// Grant items
const grant = await postApiItemGrant({
  body: {
    endUserId: 'user-42',
    grants: [
      { definitionId: 'gold-uuid', quantity: 1000 },
    ],
    source: 'admin_grant',
  },
});

// Read balance
const balance = await getApiItemUsersByEndUserIdBalanceByKey({
  path: { endUserId: 'user-42', key: 'gold' },
});

// Read full inventory
const inventory = await getApiItemUsersByEndUserIdInventory({
  path: { endUserId: 'user-42' },
});
```

### Exchange module

```typescript
// Create an exchange config and option
const exchangeConfig = await postApiExchangeConfigs({
  body: {
    name: 'Gem Exchange',
    alias: 'gem_exchange',
    isActive: true,
  },
});

await postApiExchangeConfigsByConfigKeyOptions({
  path: { configKey: 'gem_exchange' },
  body: {
    name: '100 gold → 10 gems',
    costItems: [{ definitionId: 'gold-uuid', quantity: 100 }],
    rewardItems: [{ definitionId: 'gem-uuid', quantity: 10 }],
    userLimit: 5,
    globalLimit: 1000,
    sortOrder: 1,
    isActive: true,
  },
});

// Execute exchange
const exchangeResult = await postApiExchangeOptionsByOptionIdExecute({
  path: { optionId: 'option-uuid' },
  body: {
    endUserId: 'user-42',
    idempotencyKey: 'unique-id',
  },
});
```

### Client credentials

```typescript
// Create a credential pair
const cred = await postApiClientCredentials({
  body: {
    name: 'Production Keys',
    expiresAt: '2027-01-01T00:00:00.000Z',
  },
});
// cred.data.secret is visible only here — keep it safe.

// List credentials
const creds = await getApiClientCredentials();

// Rotate
const rotated = await postApiClientCredentialsByIdRotate({
  path: { id: cred.data.id },
});
```

## Function naming rules

SDK function names are built from HTTP verb + path:

| HTTP request | SDK function |
|--------------|-------------|
| `GET /api/check-in/configs` | `getApiCheckInConfigs()` |
| `POST /api/check-in/configs` | `postApiCheckInConfigs()` |
| `GET /api/check-in/configs/{key}` | `getApiCheckInConfigsByKey()` |
| `PATCH /api/check-in/configs/{id}` | `patchApiCheckInConfigsById()` |
| `DELETE /api/check-in/configs/{id}` | `deleteApiCheckInConfigsById()` |
| `POST /api/item/grant` | `postApiItemGrant()` |
| `GET /api/item/users/{endUserId}/balance/{key}` | `getApiItemUsersByEndUserIdBalanceByKey()` |

## Parameter shape

Every function accepts a single `options` object:

```typescript
await someEndpoint({
  path: { key: 'value' },           // path params
  query: { definitionId: 'uuid' },  // query params
  body: { ... },                    // request body
  headers: { ... },                 // extra headers
});
```

## OpenAPI spec

The full OpenAPI 3.1.0 spec is at:

```
packages/sdk-core/specs/openapi-admin.json
```

A running server also exposes it at `GET /openapi.json`, with a Scalar UI at `GET /docs`.
